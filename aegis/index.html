03 Modules

Tam Prescriptive Stack Full Prescriptive Stack

Prediction'dan audit'e kadar birlikte çalışan altı özel modül. Six specialized modules working as one coherent system.

Prediction

Kalibre Ensemble Calibrated Ensemble

XGBoost, LightGBM ve CatBoost — Platt scaling ile kalibre edilmiş churn olasılığı. Class imbalance, eksik değer ve yüksek kardinaliteli kategorikler kutu dışı çalışır. XGBoost, LightGBM, and CatBoost with Platt scaling calibration. Handles class imbalance, missing values, and high-cardinality categoricals out of the box.

Preprocessing

Polars Pipeline

Polars tabanlı ön işleme. Feature state train sırasında persist ediliyor, serve sırasında yeniden yükleniyor — train/serve tutarsızlığı yok. Polars-based preprocessing with persisted feature state. Loaded at serve time to guarantee train/serve consistency — no silent feature drift.

Explanation

SHAP + DiCE

SHAP TreeExplainer ile global ve lokal feature önem analizi. DiCE ile iş kısıtlarına uyan minimum değişiklik counterfactual'ları. "Bu müşteri neden churn eder?" + "Ne değişse kalır?" SHAP TreeExplainer for global and local feature importance. DiCE counterfactuals with business constraints for minimal-change interventions. "Why will this customer churn?" + "What would make them stay?"

Financial

Beklenen Değer Kararı Expected Value Framework

Churn olasılığı, CLTV, aksiyon maliyeti, margin ve WACC'yi birleştiren NPV tarzı karar motoru. Müşteri başına retention ROI'sini dolar cinsinden hesaplar — eşiği geçmeyene aksiyon önerilmez. NPV-style decision engine combining churn probability, CLTV, action cost, margin, and WACC discounting. Converts probability into dollar-denominated retention ROI per customer — no intervention recommended below threshold.

Causal

T-Learner CATE

T-Learner tedavi etkisi tahmini. CATE = P(churn | tedavi) - P(churn | kontrol). Sleeping dog tespiti (aksiyon churn'ü artıranlar). Feature leakage guard, per-arm minimum satır zorunluluğu, treatment evidence metadata. T-Learner treatment effect estimation. CATE = P(churn|treatment) - P(churn|control). Sleeping dog detection. Feature leakage guards, per-arm row minimums, treatment evidence metadata in every API response.

Audit & Auth

Denetlenebilir Altyapı Auditable Infrastructure

JWT + RBAC, request-scoped tenant context, rate limiting. Privacy-safe JSONL audit olayları — hassas alanlar redacted. Immutable policy snapshot + digest. Artifact registry ile pilot-grade replay kanıtı. JWT auth with RBAC scopes, request-scoped tenant context, rate limiting. Privacy-safe JSONL audit events with field redaction. Immutable policy snapshots with digests. Artifact registry for pilot-grade replay evidence.

Dürüst Bir Not — CATE ve Dataset Honest Note — CATE and Dataset

Mevcut demo IBM Telco dataset'ini kullanıyor. Bu dataset gerçek tedavi/kontrol ataması içermiyor; uplift akışı simüle edilmiş etiketlerle çalışıyor. Sistem bunu açıkça API response'unda bildiriyor (treatment_source: "simulation"). Gerçek nedensel CATE için randomize campaign verisi gerekiyor. Güçlü hedef dataset: KKBox (büyük ölçekli temporal churn), Criteo Uplift, Hillstrom. The current demo runs on the IBM Telco dataset, which has no real treatment/control assignment. The uplift flow uses simulated labels and reports this explicitly in every API response (treatment_source: "simulation"). Real causal CATE requires a randomized campaign dataset. Target datasets: KKBox (large-scale temporal churn), Criteo Uplift, Hillstrom.
