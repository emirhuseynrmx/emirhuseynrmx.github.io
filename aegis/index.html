03 Modules

Full Prescriptive Stack

Six specialized modules working as a unified pipeline.

Prediction

XGBoost Ensemble

Calibrated churn prediction with Platt scaling. Handles class imbalance, missing values, and high-cardinality categoricals out of the box.

Explanation

SHAP TreeExplainer

Model interpretation and causal analysis via DoWhy. Global and local feature importance with interaction effects.

Prescription

DiCE Counterfactuals

Actionable prescriptions with business constraints. Generates minimal-change interventions that respect feasibility bounds.

Financial

Expected Value Framework

NPV-style decisioning with WACC discounting. Converts churn probability into dollar-denominated retention ROI.

Causal

T-Learner CATE

Treatment effect estimation and sleeping dog detection. Identifies customers who would churn because of intervention.

Validation

Backtesting Engine

PSI drift monitoring and temporal validation. Walk-forward testing ensures model stability across time periods.
