Ne yapar? What it does

🌐

Encoding tespitiEncoding detection

chardet ile UTF-8, Latin-1, Windows-1254 ve diğer encoding'leri otomatik algılar. Bozuk karakter sorunu biter. Detects UTF-8, Latin-1, Windows-1254 and other encodings automatically via chardet. Broken character problems end.

🔢

Tip çıkarımı ve dönüşümType inference & conversion

Sayı gibi görünen string'leri, tarih gibi görünen metinleri otomatik dönüştürür. Pandas 3.x StringDtype uyumlu. Auto-converts strings that look like numbers, text that looks like dates. Compatible with Pandas 3.x StringDtype.

🗑️

Duplicate kaldırmaDuplicate removal

Tam eşleşme ile tekrarlayan satırları bulur. İstersen sadece raporlar, istersen siler. Finds repeating rows by exact match. Reports only, or deletes — your choice.

📄

Excel okumaExcel reading

openpyxl ile çok sayfalı Excel dosyaları. Hangi sayfa, hangi satırdan başla, hangi sütunlar — parametre ile. Multi-sheet Excel files via openpyxl. Which sheet, which start row, which columns — all via parameters.

📁

Çoklu exportMulti-format export

Temizlenen veriyi CSV, Excel, JSONL veya Parquet olarak kaydet. Aynı pipeline, farklı çıktı. Save cleaned data as CSV, Excel, JSONL, or Parquet. Same pipeline, different output.

💻

CLI arayüzü

Script veya terminal üzerinden doğrudan çalıştırılabilir. Otomasyon pipeline'larına kolayca eklenir. Runnable directly from script or terminal. Easily added to automation pipelines.
