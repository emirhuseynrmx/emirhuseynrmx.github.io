matplotlib ← Plotly Plotly interactive, tarayıcıda güzel. Ama amaç PDF raporu — PDF'e interaktif grafik gömülemez. matplotlib statik PNG üretir, reportlab bu PNG'yi yüksek çözünürlükte PDF'e yerleştirir. Plotly için kWeasyPrint veya Kaleido ek bağımlılık gerekir, risk artar. matplotlib tek bir pip install.Plotly is interactive, looks great in browser. But the goal is PDF reports — you can't embed interactive charts in PDF. matplotlib produces static PNG, reportlab places this PNG into PDF at high resolution. Plotly needs WeasyPrint or Kaleido as extra dependencies, increasing risk. matplotlib is a single pip install.
veri hazırlama Ham trade log'u birçok sorun içeriyor: tarih sütunu string ("2024-01-15 09:32" format), fiyat sütunları bazen None, aynı trade_id'ye çift kayıt (partial fills). pd.to_datetime() + ffill + groupby trade_id aggregate. df.diff() ile günlük PnL farkı hesaplanıyor.Raw trade log has many issues: date column as string ("2024-01-15 09:32" format), price columns sometimes None, duplicate trade_id (partial fills). pd.to_datetime() + ffill + groupby trade_id aggregate. Daily PnL delta calculated with df.diff().
Sharpe hesabı Sharpe = (mean_daily_return − rf_rate) / std_daily_return × √252. Yıllık olarak normalize etmek için √252 çarpımı (252 işlem günü). rf_rate müşteri parametreden giriyor (varsayılan: 0.04 / 252 günlük). Sortino sadece negatif günlerin std'sini kullanır.Sharpe = (mean_daily_return − rf_rate) / std_daily_return × √252. Multiplied by √252 to annualize (252 trading days). rf_rate comes from client parameter (default: 0.04 / 252 daily). Sortino uses only std of negative days.
