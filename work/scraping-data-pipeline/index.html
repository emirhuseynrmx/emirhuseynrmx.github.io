Async scrapingAsync scraping

httpx + asyncio ile eşzamanlı HTTP istekleri. Tek sayfa için de, bin sayfa için de aynı hızda. Concurrent HTTP requests with httpx + asyncio. Same speed whether you scrape one page or a thousand.

🔧

Config-drivenConfig-driven

Hedef URL'ler, CSS selectorlar, retry sayısı ve export formatı — hepsi JSON config'de. Kod açmadan değiştirilir. Target URLs, CSS selectors, retry count, and export format — all in JSON config. No code changes needed.

🛡️

Pydantic v2

Her kayıt Pydantic modelden geçer. Tip hatası, eksik alan, beklenmedik veri — hepsi net hata mesajıyla yakalanır. Every record passes through a Pydantic model. Type errors, missing fields, unexpected data — all caught with clear error messages.

📊

Pandera

DataFrame'e geçmeden önce Pandera schema doğrulaması. Beklediğin veriden sapma olursa pipeline durur. Pandera schema validation before the DataFrame is returned. If data drifts from what you expect, the pipeline stops.

📁

Çoklu formatMulti-format export

CSV, Excel, JSONL ve Parquet desteği. Aynı config, farklı format. CSV, Excel, JSONL, and Parquet support. Same config, different format.

🔁

Retry mantığıRetry logic

Başarısız istekler otomatik olarak yeniden denenir. Timeout, 5xx ve network hatası — hepsini ele alır. Failed requests are automatically retried. Handles timeouts, 5xx errors, and network failures.
