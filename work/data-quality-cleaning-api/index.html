Ne yapar? What it does

🔍

Otomatik tip çıkarımıAuto type inference

Sütun tiplerini elle belirtmene gerek yok. API string, sayı, tarih ve boolean'ı kendisi tespit eder. No need to specify column types manually. The API detects strings, numbers, dates, and booleans itself.

🧹

Eksik değer doldurmaMissing value imputation

Sayısal sütunlar için medyan, kategorik için mod. Ya da özel bir doldurma stratejisi parametre olarak geçilebilir. Median for numeric columns, mode for categorical. Or pass a custom imputation strategy as a parameter.

👥

Duplicate tespitiDuplicate detection

Tam eşleşme ve fuzzy match seçenekleriyle tekrarlayan satırları bulur ve raporlar. Silme veya işaretleme yapılabilir. Finds repeating rows with exact-match or fuzzy-match options. Can delete or flag duplicates.

📋

Kalite raporuQuality report

Her response'ta: eksiklik oranı, tip dağılımı, anomali sayısı, temizlenen satır sayısı. Every response includes: null rate, type distribution, anomaly count, cleaned row count.

⚙️

Pandera Schema

Output veri her zaman Pandera schema'dan geçer. Temizledikten sonra bile beklenen formatta olduğu garanti altında. Output data always passes through a Pandera schema. Even after cleaning, the expected format is guaranteed.

🔌

Kolay entegrasyonEasy integration

JSON veya multipart/form-data ile CSV yükle. Temiz veriyi JSON veya CSV olarak geri al. Ekstra kütüphane gerekmez. Upload CSV via JSON or multipart/form-data. Get clean data back as JSON or CSV. No extra libraries needed.
