Ne yapar? What it does

🔍

/v1/csv/profile — sütun profili/v1/csv/profile — column profiling

CSV metnini gönder, her sütun için tip çıkarımı + eksik değer sayısı + istatistik özeti al. Maksimum 1.000.000 karakter CSV veya 5.000 kayıt. İdempotency: X-Idempotency-Key başlığıyla aynı istek 256 girişlik önbellekten döner. Send CSV text, get type inference + missing count + stat summary per column. Max 1,000,000 char CSV or 5,000 records. Idempotency: same request with X-Idempotency-Key header returns from 256-entry cache.

🧹

/v1/records/clean — kayıt temizleme/v1/records/clean — record cleaning

Trim, e-posta küçük harf, E.164 telefon normalizasyonu, formül nötrleştirme (=CMD gibi injection önleme), anahtar alanlara göre dedup. Maksimum 2.000 kayıt/batch. Her istek için /ops/audit'te denetim kaydı. Trim, lowercase emails, E.164 phone normalization, formula neutralization (=CMD injection prevention), dedup by key fields. Max 2,000 records/batch. Audit log entry in /ops/audit for every request.

📧

/v1/email/normalize — e-posta doğrulama/v1/email/normalize — email validation

5 hardcoded tek kullanımlık e-posta domain'i (mailinator, guerrillamail ve diğerleri) reddeder. Geçerli e-postayı normalize eder. Disposable domain listesi genişletilebilir config ile. Rejects 5 hardcoded disposable email domains (mailinator, guerrillamail, and others). Normalizes valid emails. Disposable domain list extensible via config.

📋

/ops/audit — denetim kuyruğu/ops/audit — audit tail endpoint

Her API isteğinin denetim kaydını döner: endpoint, istek boyutu, temizlenen kayıt sayısı, zaman damgası. Üretim ortamında RapidAPI trafiğini izlemek için tasarlandı. Returns audit log for every API request: endpoint, request size, records cleaned, timestamp. Designed to monitor RapidAPI traffic in production.

🔑

RapidAPI auth + rate limiterRapidAPI auth + rate limiter

X-RapidAPI-Proxy-Secret başlığı doğrulaması. Sabit pencere rate limiter: 120 istek/60 saniye. Aşıldığında 429 döner. RapidAPI marketplace'e listeleme için hazır — yerel kullanım için değil. X-RapidAPI-Proxy-Secret header validation. Fixed-window rate limiter: 120 req/60s. Returns 429 when exceeded. Ready for RapidAPI marketplace listing — not just local use.

/v1/phone/normalize + /v1/domain/parse/v1/phone/normalize + /v1/domain/parse

Telefon numarasını E.164 formatına çevirir (uluslararası ön ek dahil). Domain'i bileşenlerine ayırır: subdomain, apex, TLD, kayıtlı mı değil mi. JSON veya CSV çıktı. Converts phone number to E.164 format (including international prefix). Parses domain into components: subdomain, apex, TLD, registered or not. JSON or CSV output.

Problem neydi? Neden REST API? What was the problem? Why a REST API?

Müşteri sorunuClient problem Ajans, 5 farklı müşteriden veri alıyor: biri Excel, biri API JSON, biri Salesforce export, biri Google Sheets CSV, biri PostgreSQL dump. Her proje başında aynı temizleme adımları tekrar yazılıyor. Toplam haftalık kayıp: 8+ saat.Agency receives data from 5 different clients: one Excel, one API JSON, one Salesforce export, one Google Sheets CSV, one PostgreSQL dump. Same cleaning steps rewritten at the start of every project. Weekly waste: 8+ hours.
API ← kütüphane Neden kütüphane değil? Python kütüphanesi yalnızca Python projeleri çağırabilir. REST API dil bağımsız: n8n workflow'u, Google Sheets Apps Script, Retool dashboard, hatta müşterinin Excel makrosu POST atabilir. Tek kod, herkes kullanır.Why not a library? A Python library can only be called from Python projects. REST API is language-agnostic: n8n workflow, Google Sheets Apps Script, Retool dashboard, even the client's Excel macro can POST to it. One codebase, everyone uses it.
otomatik tip çıkarımı pd.read_csv() varsayılan olarak her şeyi string okur — tipler elle belirtilmeden. Bu API: sütunu inceler, %90+ değeri float olarak parse edilebiliyorsa → float, "2024-01-15" formatına uyuyorsa → datetime, "true"/"false" içeriyorsa → bool. Manuel schema gereksiz.pd.read_csv() by default reads everything as string — types must be specified manually. This API: inspects each column, if 90%+ values can be parsed as float → float, if they match "2024-01-15" format → datetime, if they contain "true"/"false" → bool. No manual schema needed.
Pandera son adım Temizlendikten sonra da output schema doğrulaması yapılır. Neden? Temizleme süreci beklenmedik sonuç üretebilir (medyan imputation negatif değer çıkardıysa?). Pandera bunu yakalar — downstream ML pipeline'ına kirli veri geçmez.Schema validation runs on the output after cleaning too. Why? The cleaning process can produce unexpected results (what if median imputation produced a negative value?). Pandera catches this — no dirty data reaches the downstream ML pipeline.

  • Kütüphane değil API: müşteri Python bilmeden entegre eder. pip install gerektirmez, versiyon çakışması olmaz. HTTP bilen herkes kullanabilir.API not library: client integrates without knowing Python. No pip install, no version conflicts. Anyone who knows HTTP can use it.
  • FastAPI → Litestar geçişi: idempotency, rate limiting, audit log middleware korundu. Sadece router katmanı değişti. Dış API davranışı aynı.FastAPI → Litestar migration: idempotency, rate limiting, audit log middleware preserved. Only router layer changed. External API behavior unchanged.
