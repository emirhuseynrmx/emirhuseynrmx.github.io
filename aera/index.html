01 3 Ajan Pipeline3-Agent Pipeline

Planner → Executor → Critic Planner → Executor → Critic

Her ajan bir sonrakini doğrular ve geliştirir. Tek seferlik LLM çağrısı değil — iteratif, eleştirel bir süreç. Each agent validates and refines the previous one. Not a single LLM call — an iterative, critical process.

AGT 01
Planlayıcı Planner

Veriyi tarar, analiz stratejisini belirler, hangi modüllerin çalışacağına karar verir. Scans data, defines analysis strategy, decides which modules to activate.

→
AGT 02
Yürütücü Executor

Tahmin, anomali, teşvik eşleştirme ve PDF oluşturmayı yürütür. Runs forecasting, anomaly detection, incentive matching, and PDF generation.

→
AGT 03
Eleştirmen Critic

Çıktıyı iş mantığı ve tutarlılık açısından doğrular, gerekirse revize eder. Validates output for business logic and consistency, revises if needed.

02 ÖzelliklerFeatures

Kapsamlı Finansal Analiz Comprehensive Financial Analysis

KOBİ'ler için finansal karar almayı hızlandıran altı çekirdek modül. Six core modules that accelerate financial decision-making for SMEs.

Forecasting

Holt Projeksiyonu Holt Projection

Üstel yumuşatma ile trend ve mevsimsellik tahmini. 3-6-12 aylık nakit akışı projeksiyonları, güven aralıklarıyla birlikte. Exponential smoothing for trend and seasonality forecasting. 3-6-12 month cash flow projections with confidence intervals.

Anomaly

Anomali Tespiti Anomaly Detection

Z-score ve IQR tabanlı anomali tespiti. Beklenmedik gider artışlarını ve gelir düşüşlerini otomatik işaretler, bağlam açıklamasıyla sunar. Z-score and IQR-based detection. Automatically flags unexpected expense spikes and revenue drops with context explanations.

Incentives

KOSGEB / TÜBİTAK

Şirket profilinize uygun KOSGEB, TÜBİTAK ve sektörel teşvik programlarını otomatik eşleştirir. Başvuru gereksinimlerini özetler. Automatically matches KOSGEB, TÜBİTAK, and sectoral incentive programs to your company profile. Summarizes application requirements.

Report

Board-Ready PDF Board-Ready PDF

Tek tıkla profesyonel A4 PDF raporu. Grafikler, tablolar, yönetici özeti. Gemini 2.5 Flash ile doğal dil yorumlama eklenmiş. One-click professional A4 PDF with charts, tables, and executive summary. Gemini 2.5 Flash adds natural language interpretation.

Demos

23 Sektör Demo 23 Industry Demos

Perakende, üretim, teknoloji dahil 23 sektör için hazır demo dataset'leri ve analiz şablonları. Sıfırdan başlamak zorunda değilsiniz. Ready-made demo datasets and templates for 23 industries including retail, manufacturing, and technology. No need to start from scratch.

Input

CSV / XLSX

CSV veya Excel dosyası yükle, geri kalanını Polars tabanlı pipeline yönetir. Şema doğrulama, otomatik tip çıkarımı, eksik değer işleme. Upload CSV or Excel — Polars-based pipeline handles the rest. Schema validation, automatic type inference, missing value handling.

03 Technology

Sağlam Altyapı Robust Infrastructure

Rust + Axum backend, Polars veri motoru, Gemini 2.5 Flash LLM entegrasyonu. Rust + Axum backend, Polars data engine, Gemini 2.5 Flash LLM integration.
